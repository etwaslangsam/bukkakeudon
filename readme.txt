■操作方法

マウス
	カーソルが変化しない場所
		クリック：ランダムトーク
		ダブルクリック：ゴーストメニュー表示
		
	カーソルが変化する場所
		クリック：さわり反応
		左右に動かす：なで反応

キーボード
	mキー：ゴーストメニュー表示
	tキー：ランダムトーク
	cキー：コミュニケーション
	pキー：写真撮影のお願い

	xキー：着せ替え時に反応しなくなった場合の修復
	
ゴーストメニュー
	ぶっかけさせて：ぶっかけモード
	
	話を聞く：ランダムトーク
	話しかける：コミュニケーション
	
	愛称をつける：彼女にニックネームをつける
	呼び方を変えて：彼女からの呼ばれ方を変える
	
	自由に話して：４５〜１３５秒の間で自動的にランダムトークするようになる
	なにも言わないで：自動的にランダムトークしないようにする

	時報のオン/オフ：１時間毎に現在時を告知する/しない

	常に手前に：他のウインドウより前に表示
	奥に沈む：トーク後に他のウインドウの裏に隠れる
